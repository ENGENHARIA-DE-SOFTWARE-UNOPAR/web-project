# JavaScript 
Linguagem tipada 
### Tipos de dados em JS
* Primitivos
* Boolean
* Null
* Undefined
* Number
* String

### Escopo e declaração
* Escopo Global
* Ecopo de função
- variavel de bloco: criada par uso dentro de instruções(for, while, if)

### Operadores matemáticos
* + Soma
* - Subtração
* ** Raiz quadrada
* / divisão
* % mod
* e Outros

### Operadores lógicos
* && AND (E)
* || OR (OU)
* ! NOT (Negação) 


### Operadores relacionais
* == igual
* === igual ou do mesmo tipo
* != Diferente
* !== Diferente e de tipos diferentes
* > Maior que
* < Menor que
* >= Maior ou igual
* <= Menor ou igual
